How did information go from a textarea in hero-detail to the mock-heroes?

  • in mock-heroes.ts, list out the mock heros object
  • in heroes.html, use ngFor to list out all the heros
  • in heroes.html, add (click) listener to see which hero is clicked
  • in heroes.ts, write function to set the clicked hero to be the selected hero
  • in heroes.html, render hero-details, and pass in selected hero as an [input]
  • in hero-detail.ts, receive @Input from heroes.html
  • in hero-detail.html, use ngModel 2 way binding to set the @Input property
